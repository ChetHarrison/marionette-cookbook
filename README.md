marionette.Recipes
===================

Examples of common patterns with Marionette.js

### Browser Support


#### Getting Started
+ `git clone git@github.com:Puppets/marionette-recipes.git`
+ ` cd marioonette-recipes`
+ `bower install`
+ `python -m SimpleHTTPServer 3000`
+ open `localhost:3000`

#### Examples

+ Tabs
+ Lists
  + Simple list
  + sorted list
+ Progress Bars
  + Simple Progress Bar
  + NProgress Bar
